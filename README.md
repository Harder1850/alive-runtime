# ALIVE Runtime

> **Constitution defines → Runtime governs → Mind thinks → Body acts → Interface displays**

## Purpose
Governance and orchestration layer. Controls flow, enforces law, and gates cognition and action.

## Responsibilities
- STG (Stop Thinking Gate)
- Routing and signal flow
- Scheduling and lifecycle
- Enforcement of invariants
- Action validation
- Priority and interrupt handling

## Authority
- Enforces Constitution
- Governs Mind
- Directs Body

## Core Principle
Runtime decides: when thinking occurs, when actions are allowed, how signals move.

## Non-Scope
- No cognition
- No long-term memory
- No direct execution
- No external interfacing

## Drift Warning
⚠️ If runtime begins thinking or storing memory, boundaries are broken.
