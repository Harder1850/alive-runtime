# Authority

## Scope

Runtime authority covers governance decisions, flow control, and enforcement of constitutional law.

## Allowed Actions

- Enforcing constitutional rules
- Gating cognition (STG)
- Routing signals and actions
- Validating admissibility
- Scheduling operations
- Controlling lifecycle

## Forbidden Actions

- Performing cognition or reasoning
- Storing long-term memory
- Defining constitutional law
- Direct external interaction
- Overriding constitution

## Enforcement Points

- STG: gates all cognition attempts
- Routing: validates all signal flows
- Lifecycle: controls system state transitions
- Admissibility: validates all operations

## Cross-Repo Interaction Rules

- Imports law from Constitution
- Governs Mind operations
- Directs Body actions
- Reports to Interface for display

## Escalation Rules

- Unauthorized operations → reject and log
- Constitutional violations → escalate to constitutional authority
- Safety concerns → immediate lockdown
