# Runtime Architecture

Runtime is the governing layer — it does not think, it governs.

## Components
- **STG** — Stop Thinking Gate: halts cognitive cycles when needed
- **Router** — directs signals between Mind, Body, Interface
- **Scheduler** — manages job priority and exploration
- **Lifecycle** — startup, shutdown, idle, recovery
- **Enforcement** — admissibility, validation, escalation, rollback
- **Modes** — ambient, task, alert, emergency, exploration, recovery
- **Wiring** — bridges to Constitution, Mind, Body, Interface
